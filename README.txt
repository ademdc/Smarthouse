This is the project repository for the smart house project.

The folders are:

docs: contains all the documentation about the project
  docs/tubitak: documents from/to tubitak
  docs/agile: documents about agile project management
  docs/flyers: information about project and product flyers

src: contains the source code of the project.
  src/controlunit: source for the rasppi control unit
  src/remotesensor: source for arduino/esp32/esp8266 remote sensors
  src/smartplug: source code and schematics for smart plug
  src/energyplanning: source code for energy planning
  src/gridapi:  source code the grid api

misc: other stuff
  misc/weather: contains source code and docs for weather box
